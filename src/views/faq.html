FAQ

How do I use Ano.js?

Ano.js animations can be used by copy-pasting two lines code that we've already optimized for any website. There are more detailed steps on our About page.

What do I need to become a collaborator?

Absolutely nothing. Just become a collaborator first. All the materials and resources you need to be a collaborator are in the GitHub repository. Learn as you go, and don't be afraid to ask fellow collaborators for help.

Are there job positions on staff?

There are none at the moment. But, this is open to change. In the future, we may look for a Gitmaster, a technical lead, and a social media marketer.

If you couldn't find the answer you were looking for, please feel free to contact us!
