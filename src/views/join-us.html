Join Us!

Become a Collaborator

We're always looking for developers to help improve Ano.js. Ano.js is powered by open source contributions, so the best way is to become a collaborator on this project, and that can be done just by filling out the form below!

Benefits of being a Collaborator
  • Paste it on your LinkedIn
  • Being an open source contributor is a great work experience to show
  • Add to your portfolio
  • Show off Ano.js to all your future employers
  • Put it on your resume
  • No time constraints or commitments
  • You can choose how much time you dedicate to this project
  • Receive recognition
  • Collaborators with the most amount of contributions are put higher on the top collaborators list
  • Your animations will be listed on our animations page labelling you as the creator
  • More contributions = more recognition
  • Gain experience
  • Great way to learn more about software engineering
  • Network with fellow collaborators
Collaborator Qualifications

Don't worry if you don't have all of the qualifications. Just be a collaborator and learn as you go!

  • HTML5 Canvas
  • Basic Javascript
  • Optimization techniques
  • More information on this on the GitHub repository
  • Basic Git/GitHub skills
  • Cloning, pulling, pushing
  • Pull Requests
Get Invited!

After submitting this form, you will receive an invitation in your email to become a collaborator on the Ano.js Animations GitHub repository.

If you do not get an invite, please double check that you have typed in your GitHub username correctly.

NOTE: Make sure you're logged in before you accept the invitation.
