How to use Ano.js

First, you need to select which animation you would like to use. On our animations page, there is a gallery of animation demos to check out.

Second, you must receive the code for the animation. After clicking on the animation, a code popup window will show. Copy the code inside of the gray-colored box. Instructions will be posted there as well.

Finally, take the code and paste it into the body tag of your HTML file, or wherever you'd like in your website. Pull it up in a browser and the animation should be working!
